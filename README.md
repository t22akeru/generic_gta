# Generic GTA Browser

## ゲームの概要
ブラウザ上で動く **3D アクションゲーム**です。Three.js を利用し、オープンワールドを再現します。プレイヤーは街中を自由に走り回り、車両や武器を駆使してタスクを完了することが目的です。

## ゲームの流れ
1. **探索** – 街道・建物・車両・電車を移動し、インタラクティブなオブジェクトを発見します。
2. **アクション** – 左クリックで武器発射。左下げた走行や車両乗り降りが可能です。
3. **AI 追跡・対決** – 警察車両や敵NPCはプレイヤーを追跡し、適度に攻撃します。
4. **昼夜サイクル** – 時間経過で昼夜が自然に切り替わります。日中は視界が明るく、夜中は照明が低減します。
5. **HUD 表示** – HP・Wanted Level・Ammo・時刻を画面右上に表示し、ゲーム状態を把握しやすいです。

## 主な特徴
- **オープンワールドの自由度** – 何処へでも移動可能。車両は `E` キーで乗り降りします。
- **リアルタイム AI 行動** – 警察・敵NPCはプレイヤーの行動を観測し、適応的に対策します。
- **武器管理** – 銈数・リロード状態を HUD で確認。弾が無くなると自動リロード。
- **昼夜サイクル** – 日中は明るく、夜中は低光量。視界に影響します。

## 操作方法
| キー | アクション |
|---|---|
| `WASD` / 矢印キー | 移動 |
| `Shift` | ダッシュ |
| `Space` | ジャンプ |
| `E` | 車両乗り降り |
| 左クリック | 射撃 |
| マウス移動 | 視点変更 |

## 起動方法
ファイル直開き（`file://`）では動作しません。ローカル HTTP サーバ経由で起動してください。

### 方法 1: スクリプト使用
```bash
./serve.sh
```
ブラウザに `http://localhost:8080` を開く。

### 方法 2: Python HTTP サーバ
```bash
python3 -m http.server 8080
```
同様 URL をブラウザで開く。

## 開発メモ
- Three.js は CDN 経由読み込まる。ローカルパッケージ不要。
- ゲームロジックは `js/` 配下に配置。
- デバッグ時は Chrome DevTools のコンソールを確認。HUD は Canvas 上へ描画。
