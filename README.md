# Generic GTA Browser

ブラウザで動く、GTA風の3Dアクションゲームです。Three.js を使って、街・車・歩行者・電車・警察追跡・昼夜サイクルを表現しています。

## 特徴

- 昼夜の時間変化
- 手配度システム
- 歩行者・車両・警察の移動
- 左クリック射撃
- 武器の所持と弾数管理
- 電車、駅、街灯、建物の発光表現
- HP / Wanted / Ammo / 時刻 の HUD

## 操作方法

- `WASD` / 矢印キー: 移動
- `Shift`: ダッシュ
- `Space`: ジャンプ
- `E`: 車の乗り降り
- 左クリック: 射撃
- マウス: 視点移動

## 起動方法

このゲームはローカルサーバー経由で開いてください。`file://` 直開きでは動作しません。

### 方法1: 付属のスクリプトを使う

```bash
./serve.sh
```

起動後、ブラウザで以下を開きます。

```text
http://localhost:8080
```

### 方法2: Python で起動する

```bash
python3 -m http.server 8080
```

その後、ブラウザで以下を開きます。

```text
http://localhost:8080
```

## 開発メモ

- Three.js は CDN から読み込んでいます。
- メインの入口は `index.html` です。
- ゲームロジックは `js/` 配下にあります。
